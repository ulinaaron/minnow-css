Grid

The Minnow grid is small yet highly powerful. It is responsive and can be used in full-width fluid layouts. If you would like to have a fixed-width container you can use the.mw-wrapperclasses.

Columns can also be nested but the inner-gutters will not be cleared automatically.
Class Description
.mw-row Add this class to a<div> element to define the row. Necessary for all columns.
.mw-row.gutterless Clears the left and right padding on the first and last columns.
